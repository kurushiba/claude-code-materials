Claude Code の特長的な機能

6 / 7
🧠
100万トークン対応
プロジェクト全体を「まるごと頭に入れて」から作業。依存関係の見落としが起きにくい
🖼️
自然な入力受付
スクリーンショット・エラーログ・URL をそのまま貼り付けるだけでOK
📋
Plan Mode（計画モード）
「いきなり実装する前に設計図を見せて」と言える機能。手戻りを防げる
📄
CLAUDE.md
プロジェクトのルールブック。命名規則・方針を書くと毎セッション自動で読み込まれる
🪝
Hooks（フック）
「ファイル編集後に自動フォーマット」など強制力ある自動化を設定できる
⚡
Skills / サブエージェント
スラッシュコマンドで定型作業を一発実行。複数エージェントの並列処理も可能
